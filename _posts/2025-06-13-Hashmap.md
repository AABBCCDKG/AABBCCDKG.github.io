[242. Valid Anagram](https://leetcode.com/problems/valid-anagram/description/)


```python
class Solution:
    def validAnagram(self, s: str, t: str) -> bool:
        hashmap = {}
        for char in s:
            hashmap[char] += hashmap.get(char, 0)
        
        for char in t:
            if char not in hashmap:
                return False
            hashmap[char] -= hashmap.get(char, 0)
            if hashmap[char] < 0:
                return False
        return True
```
