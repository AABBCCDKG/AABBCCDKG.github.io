# Linear Regression

## 1.1 What is Linear Regression

假设因变量`y`和自变量`x`之间是线性关系，用数据找到最优参数(The parameters are estimated by minimizing the sum of squared residuals)

Model: $$y = X\beta + \varepsilon$$

- $y$: 目标变量 (target variable)
- $X$: 特征矩阵 (feature matrix)
    - 行是样本，列是特征 (rows are samples, columns are features)
- $\beta$: 真实的参数(只有上帝知道，我们永远无法知道)
- $X\beta$: 特征 X 系数，模型能解释的部分 (The part which can be explained by the model)
- $\varepsilon$: 模型解释不了的部分(装修，运气，风水....不是函数的一部分，是承认模型的不完美)
    - 真实的线: $X\beta$ -> 真实的线性关系(未知的，我们永远不知道)
    - 真实的点: $y = X\beta + \varepsilon$ -> 每一个观测值都偏离真实的线一点(噪声)
    - 哪怕是理想的线，也无法描绘出所有点
- $\hat{\beta}$: 通过OLS估计出来的参数
- $\hat{y} = X\hat{\beta}$: 预测值(拟合的函数，不包括$\varepsilon$)
- $\beta_0$: 截距，完整的模型是$y = \beta_0 + \beta_1x_1 + \beta_2x_2 + \varepsilon$
    - 矩阵形式中X的第一列全是`1`对应$\beta_0$
    - $\beta_i$: 特征$i$的系数
    - $x_i$: 第$i$个特征(feature $i$)
